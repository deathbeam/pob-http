# pob-http
HTTP API for Path of Building

## building

```bash
./build.sh
```

## running

```bash
./start.sh
```

## usage

Getting PoB code

```bash
curl -s localhost:8000/code/thedeathbeam/MaldingOfTheFlesh
```

Getting JSON data

```bash
curl -s localhost:8000/thedeathbeam/MaldingOfTheFlesh
```
